git mv [file] [new file]	=> 파일 이름 변경 / git에서도 자동으로 인식

git show [commit id]	=> 특정 커밋의 상세 정보 확인

git diff head 		=> 수정 파일 커밋 전, 변경 사항 비교

git remote add [원격저장소별칭 ] [원격저장소URL]  => 원격 저장소 연결
==============

HEAD	: git에서 포인터 개념
	: 최종적인 커밋 작업의 위치 가르킴
	: 부모 커밋을 기반으로 새로운 커밋을 만듬 ≒ 부모 커밋을 가르킴
	: 최소 한 번 이상 커밋을 해야 HEAD가 존재 ( 처음 커밋에는 HEAD의 포인터 X )
	: 커밋될 때마다 한 단계씩 이동하며, 최종 시점을 의미함
